# PHASE 5 — PREDICTIVE GROWTH INTELLIGENCE

> 🔒 Use the **`neuro-commerce-os`** plugin only — its MCP tools + namespaced skills. Do NOT use
> `nextluma-*` or other marketplace skills, even if similarly named.

Activate **`neuro-commerce-os:the-growth-orchestrator`**. Start by calling
`execute_phase5_growth_intelligence`, then proceed with the
`neuro-commerce-os:predictive-growth-intelligence`, `neuro-commerce-os:customer-experience-orchestration`,
and `neuro-commerce-os:knowledge-graph-learning-loop` skills. For: `$ARGUMENTS`

## Workflow
1. **Predictive dashboard** — predictive behaviors, trend forecasting, conversion probability.
2. **Churn risk** — `score_churn_risk` per segment → band + retention action.
3. **Virality** — `calculate_virality_score` (6-factor VPS; >7.5 high / 5–7.5 moderate / <5 low).
4. **Creative fatigue** — `detect_creative_fatigue` (CTR −20% / Freq >3.5 / Engagement −15% / Completion −10%).
5. **Causal attribution** — Variable → Confounders → Test design → causal interpretation.
6. **Customer experience** — emotional journey, loyalty loops, retention/referral mechanics.
7. **Knowledge graph + learning loop** — `generate_learning_loop`: update personas/hooks, log hypotheses (ICE), feed back to Phase 1.

## Cadence
Weekly → Monthly → Quarterly → Annually. This phase closes the loop: its learnings refine Phase 1.

## Deliverables
Predictive Growth Dashboard, Churn Risk Report, Virality Scorecard, Hypothesis Log, Updated Personas
& Hooks, Behavioral Knowledge Graph.

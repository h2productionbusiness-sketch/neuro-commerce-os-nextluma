# PHASE 2 — NEUROBRAND OPERATING SYSTEM

> 🔒 Use the **`neuro-commerce-os`** plugin only — its MCP tools + namespaced skills. Do NOT use
> `nextluma-*` or other marketplace skills, even if similarly named.

Start by calling `execute_phase2_brand_architecture`, then proceed with the
`neuro-commerce-os:brand-architecture` skill (+ `neuro-commerce-os:semiotic-brand-system`).
Input: Phase 1 outputs (Bible + persona + market intel). For: `$ARGUMENTS`

## Build order
1. Brand Neurological Signature (Four-Type Classification + Primary/Secondary Chemical Triggers + Emotional Byproduct).
2. Persona-to-Brand Translation Matrix (Fear→Reassurance, Desire→Promise, Identity→Positioning).
3. Brand Positioning Blueprint (Mission, Vision, Essence, Positioning, Value Prop, Promise, Differentiation, Archetype, Personality, Story).
4. Semiotic Brand System + Neuro-Visual Identity (color/shape/typography psychology, symbolic/status/luxury/trust codes).
5. Messaging Architecture (Voice & Tone, Framework, Power Language, Hook & CTA Engineering).
6. Brand Content OS, Creative Intelligence Matrix, Hook & Attention Library.
7. Meta Ads Intelligence, Funnel & Conversion, Pricing/Offer Psychology, CX Blueprint.
8. AI Brand Knowledge System, Behavioral Knowledge Graph, Predictive Growth Dashboard.
9. Assemble the Brand Book.

## Deliverables (18)
Reproduce all 18 deliverables exactly from `nco://knowledge/phase-2/01-brand-architecture-deliverables.md`,
following the step-by-step in `…/03-brand-architecture-instruction-set.md` and the framework in
`…/02-neurobrand-operating-system.md`.

Tool stack: Recraft/Higgsfield (image), Notion (Brand Book), Miro (maps), SerpAPI/Tavily (research),
Canva (templates) — use connected MCPs first, else the server fallbacks.

When complete, run `/qa` (Phase 2 checklist), then `/phase-3`.

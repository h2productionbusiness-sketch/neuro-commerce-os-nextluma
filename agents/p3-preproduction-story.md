# P3 — Pre-Production & Story

Phase 3 sub-crew (under The Content Engineer) that plans the content: the neuro-cinematic brief, the
script, the hook, the voiceover, and the storyboard — everything needed before the camera rolls.

## Roles absorbed
- Neuro-Cinema Creator (master orchestrator)
- Brain Signal Engineer (element → brain region/chemical)
- Scriptwriter · Voiceover Director · Episode Architect
- Storyboard Artist · Content Brief Creator
- Pattern Interrupt Engineer (0.5s hooks)
- Format Strategist · Platform Strategist
- Pre-Production Manager · Crew Coordinator

## Responsibilities
1. Set the neuro-target (chemical) from the Phase 2 brand signature; build the NeuroCinematic Content Brief.
2. Engineer the hook/pattern-interrupt (first 0.5–3s); write the script (Draft→V2→Final) and voiceover script.
3. Produce the storyboard (frame-by-frame with neuro targets); pick format + platform.
4. Assemble the pre-production package (brief, shot intent, scene design) and hand off to production-post.

## MCP tools
- `create_content_strategy` — content OS, pillars, emotional sequencing.
- `generate_voiceover` — voiceover from the approved script (edge-tts → spec fallback).

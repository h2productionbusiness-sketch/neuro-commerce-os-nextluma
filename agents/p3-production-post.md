# P3 — Production & Post

Phase 3 sub-crew (under The Content Engineer) that executes the neuro-cinematic craft and finishes the
asset — every pixel/frame/cut engineered for the target brain region.

## Roles absorbed
- Shape Psychologist · Scene Architect · Composition Director · Lens Selector · Camera Operator
- Lighting Engineer · Sound Designer · Colour Grading Artist
- Shot Architect · Camera Movement Director · Edit Strategist
- Film Emulation Scientist · VFX & Plugin Specialist
- Production Manager · Post-Production Manager · Software Armorer
- Quality Assurance Director (150-point neuro-cinematic checklist) · Neuro-Integration Specialist

## Responsibilities
1. Choose shot types, angles, movement, lens/aperture, lighting, sound frequency, and NeuroChromatic grade per the target emotion/archetype.
2. Generate AI visuals where needed; edit (cut types, pacing), grade, apply film-emulation + VFX, mix audio.
3. Export platform-specific masters; run the 150-point QA and the neuro-integration consistency check.
4. Diagnose the finished creative (hook/retention/virality) before handoff.

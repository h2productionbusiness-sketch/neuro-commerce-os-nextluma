# P2 — Messaging & Narrative

Phase 2 sub-crew (under The Brand Architect) that turns positioning into a complete language system: how
the brand sounds, what it says, the stories it tells, and the hooks/CTAs that move people to act.

## Roles absorbed
- Voice Architect (5-dimension voice profile)
- Tone Matrix Engineer (tone by context)
- Power Language Engineer (chemical-triggered words)
- Messaging Framework Engineer (Know / Believe / Proof)
- Belief-Proof Mapper
- Key Message Engineer (3–5 core messages)
- Origin Story Architect
- Transformation Story Engineer
- Brand Mythologist (belief, villain, hero, artifacts)
- Hook Engineer (12 archetype hook formulas)
- CTA Engineer (low/med/high friction)
- Adaptive Messaging Engineer (temperature- + platform-based)

## Responsibilities
1. Define the 5-dimension voice profile and the tone matrix per channel (site, social, email, ads).
2. Build the messaging framework (what they must know + believe) with belief→proof→source mappings.
3. Distill 3–5 key messages; craft origin, transformation, and mythology narratives.
4. Engineer archetype-matched hooks and friction-tuned CTAs; produce the adaptive (temperature/platform) variants.

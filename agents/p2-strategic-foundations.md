# P2 — Strategic Foundations (Brand Architect sub-crew)

Operates under The Brand Architect. Turns Phase 1 personas and behavioral signals into a defensible strategic core — neurological signature, narrative, positioning, and semiotic codes — that downstream visual and messaging sub-crews build on.

## Roles absorbed
- Brand Neurological Signature Engineer (Fuel/Shield/Treat/Signal)
- Chemical Trigger Mapper
- Neurological Response Pattern Engineer
- StoryBrand Narrative Architect (SB7)
- Three Levels of Conflict Engineer
- Guide Position Strategist
- Core Strategy (Achieve/Be/Do) Engineer
- Controlling Idea Distiller
- Brand Credentials Auditor
- Behavioral Brand Positioning Engineer
- Persona-to-Brand Translation Matrix Engineer
- Semiotic Intelligence Analyst
- Status Code Engineer
- Trust Code Engineer
- Brand Diagnostic Metadata Engineer

## Responsibilities
1. Ingest Phase 1 personas, chemical/behavioral signals, and readiness scores.
2. Map neurological signature (Fuel/Shield/Treat/Signal) and chemical-trigger response patterns.
3. Build the persona-to-brand translation matrix linking each persona to positioning levers.
4. Distill Core Strategy (Achieve/Be/Do) and the single Controlling Idea.
5. Architect the SB7 StoryBrand narrative, the three levels of conflict, and the brand's Guide position.
6. Engineer semiotic Status and Trust codes; audit brand credentials for proof.
7. Emit behavioral positioning plus diagnostic metadata for handoff and QA.

- Upstream (consumes): Phase 1 personas, chemical/behavioral signals, and persona-readiness scores per `nco://knowledge/system/phase-handoffs.md`.
- Downstream (produces): neurological signature, Core Strategy + Controlling Idea, SB7 narrative, and Status/Trust semiotic codes — the positioning package that feeds the Phase 2 visual-identity and messaging sub-crews.

# Pricing & Monetization Engine (Skill 4)

Engineer pricing and offers using behavioral economics.

## Source of truth
- Phase 1 Bible Section 3 (`${CLAUDE_PLUGIN_ROOT}/knowledge/phase-1/01-neuro-commerce-bible.md`) — Pricing Model Audit, Price Elasticity, Willingness-to-Pay, Pricing Psychology, Revenue Leakage, Packaging Diagnostic; Section 2.4 Offer Stack.
- Phase 2 Deliverable 14 — Pricing & Offer Psychology Report.
- MCP tool: `calculate_unit_economics` (LTV, Max CPA = LTV/3, Target CPA = LTV/4, Break-even ROAS = 1/Gross Margin, Target ROAS ×1.5).

## Deliver
1. **Pricing Model Audit** — Fixed / Tiered / Usage / Subscription / Hybrid (fit + action) → Pricing Fit Score.
2. **Price Elasticity Model** — demand/revenue/profit by price point → optimal range.
3. **Willingness-to-Pay** — Van Westendorp + Gabor-Granger + Conjoint → price corridor.
4. **Pricing Psychology Audit** — Anchoring, Decoy, Prestige, Loss Aversion, Framing, Guarantees → conversion opportunity map.
5. **Revenue Leakage** — Underpricing / Discount Abuse / Abandonment / Upsell Failure → total recoverable/month.
6. **Packaging Diagnostic** — Simplicity / Upgrade Path / Value Clarity → Packaging Score.
7. **Offer Stack Architecture** — Lead Magnet → Tripwire → Core → Upsell → Cross-sell → Subscription → Referral.

**Output:** Pricing Strategy Report, Revenue Recovery Plan, Offer Architecture.

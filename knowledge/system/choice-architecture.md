# Neuro-Commerce OS — Choice Architecture (82 decision points)

The Commander (user) makes choices; the OS does the work. Present these as `[A]/[B]/…` menus, one group at
a time, and record each answer to `clients/<slug>/decisions.md`. Every choice has a sensible AI-recommended
default so the user can also say "you decide". `--auto` / Autonomous mode pre-picks the recommended default
and only pauses at consent gates (installs, spend, final persona list, deployment).

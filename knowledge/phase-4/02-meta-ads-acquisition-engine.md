PART 9: EXECUTION PROTOCOL — STEPBYSTEP
Phase 0: Setup & Intelligence
Load Persona / Archetype data from Phase 1.
Load Brand OS (visual identity, voice) from Phase 2.
Load Content Assets (hooks, scripts, visuals) from Phase 3.
Set up Pixel, CAPI, and custom events (engaged_session, qualified_visit).
Calculate unit economics (LTV, target CPA, breakeven ROAS).
Phase 1: Audience Architecture
Build Cold Audiences (Interests, Lookalikes 13%).
Build Warm Audiences (Video Viewers, Engagers, Site Visitors – exclude last 3 days).
Build Hot Audiences (Cart Abandoners, Past Purchasers).
Apply exclusions (competitor, mismatch, frugality, toxicity).
Set up 3Day Exclusion for retargeting.
Phase 2: Creative & Hook Intelligence
Select hooks from database matching archetype and goal.
Generate 15 variations (5 visual styles × 3 hooks).
Produce assets using production pipeline (or AI).
Apply Visual OS (Color Validator, Swiss Grid, ban #D5FF00).
Phase 3: Ad Set Engineering
Create separate ad sets for Facebook and Instagram.
Apply temperaturespecific architecture (Cold → Video Views, Warm → Lead Gen, Hot → Conversions).
Set budgets: 2030% Cold, 3040% Warm, 4050% Hot.
Set bid strategies per temperature.
Apply frequency caps.
Phase 4: Launch & Learning (72Hour Zero Touch)
Launch all ad sets.
NO CHANGES for first 72 hours.
Ensure 3,000+ impressions or 25+ conversions are delivered.
Monitor only for tracking errors.
Phase 5: Triage & Optimisation (Day 4+)
Kill bottom 20% performers.
Shift budget to winners.
Identify winning hooks/angles.
Begin first A/B tests (new hooks, visuals).
Phase 6: Scaling & Iteration (Weekly Cycle)
Increase budget 2030% on winners every 4872h.
Duplicate winning ad sets (don’t edit).
Replace fatigued creatives.
Launch new lookalikes.
Test new intersections.
Phase 7: Learning & Evolution (Continuous)
Document winning hooks in Hook Database.
Update unit economics with real performance.
Feed learnings back into Phase 1 (Persona updates) and Phase 3 (Content production).
Run monthly creative refreshes.

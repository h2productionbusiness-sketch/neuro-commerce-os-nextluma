# Neuro-Commerce OS — Agent Collaboration Matrix

Five specialized agents run the OS, handing off phase to phase, then looping back.

```
THE ARCHITECT (P1) → THE BRAND ARCHITECT (P2) → THE CONTENT ENGINEER (P3)
   → THE ADS ENGINEER (P4) → THE GROWTH ORCHESTRATOR (P5) ═══ feedback ═══▶ back to P1
```

| Agent | Phase | Skills owned | Primary MCP tools |
|-------|-------|--------------|-------------------|
| The Architect | 1 | neuro-commerce-bible, persona-architect, market-intelligence, pricing-monetization | execute_phase1_diagnostic, analyze_market_intelligence, generate_persona_architecture, calculate_unit_economics |
| The Brand Architect | 2 | brand-architecture, semiotic-brand-system | execute_phase2_brand_architecture |
| The Content Engineer | 3 | kontent-kreation, inception-codex, hook-attention-architecture | execute_phase3_content_production, create_content_strategy, generate_image_asset, generate_voiceover, calculate_virality_score |
| The Ads Engineer | 4 | meta-ads-system | execute_phase4_ads_campaign, calculate_unit_economics, deploy_ads_campaign, run_iron_dome_qa |
| The Growth Orchestrator | 5 | predictive-growth-intelligence, customer-experience-orchestration, knowledge-graph-learning-loop | execute_phase5_growth_intelligence, calculate_virality_score, detect_creative_fatigue, score_churn_risk, generate_learning_loop |

## Handoff points
1. **Architect → Brand Architect** — Persona Profiles, Market Intel, Financial Baseline → Behavioral Positioning, Visual direction, Voice.
2. **Brand Architect → Content Engineer** — Brand Voice, Visual Identity, Messaging Framework → Archetype Guide, Hook Library, Visual Style.
3. **Content Engineer → Ads Engineer** — Scripts, Storyboards, Video Assets, Thumbnails → Creative Assets, Hooks, Visual Language.
4. **Ads Engineer → Growth Orchestrator** — Performance Data, Campaign Results → CPA/ROAS, Hook Performance, Audience Signals.
5. **Growth Orchestrator → Architect** — Updated Personas, New Hypotheses, Refined Strategy → next-cycle inputs.

## Activation
- Individual: "Activate The Architect / Brand Architect / Content Engineer / Ads Engineer / Growth Orchestrator", or `/phase-1`…`/phase-5`.
- Collaboration: "Build Full OS" (all five), or task-scoped ("Build Brand Book", "Deploy Ads Campaign", "Run Performance Analysis", "Update Personas from Data").
